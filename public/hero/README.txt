Hero rotation images
====================

Drop project / unit photos in this folder named:

  01.jpg
  02.jpg
  03.jpg
  04.jpg

Recommended:
  - Landscape orientation, 1920×1080 or larger
  - Dark, premium interior or exterior shots work best
    (the hero applies a dark overlay so text stays readable)
  - JPEG or PNG, ~250–500 KB each for fast loading

To add more images: append additional files (05.jpg, 06.jpg, …) and
update the HERO_IMAGES array at the top of components/Hero.tsx.
